Access keys {{key_name}} Access Key Manage settings of access key {{key_name}} General Settings Key Name A unique key name that also forms the user login ID for all requestes authenticated by this key Description A brief description of this access key, which ideally explains why this access key was issued. Methods Restrict access to certain HTTP methods. Access is granted for all HTTP methods by default, if not HTTP method restriction was selected when the access key has been issued. HTTP GET Access key allows to read existing resources HTTP PUT Access key allows to store resources HTTP POST Access key allows to invoke operations on existing resources HTTP DELTE Access key allows to remove existing resources Paths Restrict access to certain resource types. Access is granted for all resource types by default, if no resource restriction was selected when the access key has been issued. Images Access key allows to access image resources located under /api/v1/images Pods Access key allows to access pod resources located under /api/v1/pods Elements Access key allows to access element resources located under /api/v1/elements Jobs Access key allows to access job resources located under /api/v1/jobs Metrics Access key allows to access metric resources located under /api/v1/metrics Users Access key allows to access user resources located under /api/v1/users Access Keys Access key allows to access access key resources located under /api/v1/accesskeys Save settings Revoke access key
